Adds a lists column and filtering options to the Mailster newsletter overview.

== Description ==
This plugin enhances your Mailster newsletter overview by adding a new column showing the assigned lists along with filtering capabilities.

Features:
* Shows assigned lists in a new column
* Displays subscriber count per list
* Filter newsletters by specific lists
* Shows number of newsletters using each list
* Sortable column
* Easy to use dropdown filter
* Multi-language support

The filter dropdown shows you:
* All available lists
* Number of subscribers per list
* Number of newsletters using each list
* Option to show newsletters without any list assigned

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/mailster-lists-column`
2. Activate the plugin
3. The lists column will appear in your newsletter overview
4. Use the dropdown filter above the newsletter list to filter by specific lists

== Frequently Asked Questions ==
= Does this work with Mailster 3.0? =
Yes, this plugin is compatible with Mailster 3.0 and above.

= Can I filter newsletters by multiple lists? =
Currently, you can filter by one list at a time.

= Are the subscriber counts live? =
Yes, the subscriber counts are always up to date.
